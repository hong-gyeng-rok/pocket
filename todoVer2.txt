앞으로 추가할 기능

1. 잠금 기능 [완료]
1.1. 드래그 후 객체가 선택되면 자물쇠 아이콘이 나오며 클릭 시 잠금
1.2. 잠금 시 여러 객체가 한번에 움직일 수 있도록 한개의 객체로 되도록 설정

2. 빠른 실행 기능 (Quick Connect)
2.1. 어떤 모드던 실행 될 것
2.2. 객체 hover 시 객체 사방위(상하좌우)에 파랑 원 아이콘 보여질 것
2.3. 아이콘 클릭 시 화살표 생성 모드로 진입
    2.3.1. 화살표는 검정 화살표로 생성될 것
2.4. 화살표 연결 로직
    2.4.1. 빈 공간 클릭 시: 해당 위치에서 화살표 끝남 (단순 화살표 생성)
    2.4.2. 다른 객체의 파랑 원(핸들) 클릭 시: 두 객체를 연결하는 화살표 생성
    2.4.3. 연결은 그룹화나 잠금과는 다름 (단순 시각적 연결)
2.5. 적용 대상
    2.5.1. 도형
    2.5.2. 이미지
    2.5.3. 메모지
    2.5.4. 텍스트 박스
2.6. 기능 구현 예시
    2.6.1. 객체에 마우스 hover
    2.6.2. 파란 원 아이콘 사방위에서 나옴
    2.6.3. 오른쪽에 있는 원 아이콘 클릭 -> 화살표 모드 변경, 객체 오른쪽에서 화살표 시작
    2.6.4. 다시 클릭 시 종료
        2.6.4.1. 바닥 클릭 시 -> 바닥에서 화살표 종료
        2.6.4.2. 다른 객체 아이콘 클릭 시 -> 해당 객체랑 이어지는 화살표 생성

3. 모든 객체 우선순위 설정 (layer 설정)
3.1. 최우선 순위
    3.1.1. 텍스트 박스, 메모지
3.2. 2순위
    3.2.1. PEN
3.3. 3순위
    3.3.1. 도형
3.4. 4순위
    3.4.1. 이미지

4. 텍스트 사이즈 설정
4.1. 현재 텍스트 기준으로 할 것
4.2. small, medium, large, x-large로 정해진 템플릿으로 만들 것
4.2.1. 각 사이즈 별 스타일 매칭 tailwind style 기준
    4.2.1.1. small = sm
    4.2.1.2. medium = m
    4.2.1.3. large = l
    4.2.1.4. x-large = xl
