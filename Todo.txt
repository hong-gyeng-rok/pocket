# Pocket Project Development Roadmap

## Phase 1: 무한 캔버스 엔진 (The Engine)
- [ ] Canvas 기본 세팅 및 Full Screen 렌더링
- [ ] 렌더링 루프 (requestAnimationFrame) 구현
- [ ] 카메라 상태(Pan, Zoom) 관리용 Ref 구현
- [ ] 마우스/휠 이벤트를 통한 Pan & Zoom 기능
- [ ] 무한 캔버스 확인을 위한 배경 격자(Grid) 구현

## Phase 2: 펜 드로잉 (The Pen)
- [ ] 선(Stroke) 데이터 구조 정의
- [ ] 마우스 이벤트를 이용한 드로잉 로직 구현
- [ ] Screen 좌표를 World 좌표로 변환하는 공식 적용
- [ ] 그려진 선들을 캔버스에 렌더링

## Phase 3: 도구 및 색상 제어 (The Tools)
- [ ] 툴바 UI 구현 (Tailwind CSS)
- [ ] 3색(검정, 빨강, 파랑) 색상 선택 기능
- [ ] 모드 전환 (펜 모드 vs 손 모드) 구현

## Phase 4: 메모지 레이어 (The Memos)
- [ ] 메모지 데이터 구조 정의
- [ ] Canvas 위에 HTML Overlay 레이어 생성
- [ ] 카메라 상태(Pan, Zoom)와 HTML 요소 위치 동기화
- [ ] 텍스트 입력 가능한 메모지 컴포넌트 구현
